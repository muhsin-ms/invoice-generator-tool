# 🧾 Invoice Generator Tool

A professional web application to generate clean and customizable invoices.

##  Features

- Add client details
- Multiple item support
- Automatic total calculation
- Tax and discount support
- Print and download invoice
- Clean business UI

## 🌐 Live Demo
https://invoice-generator-tool-one.vercel.app/

##  Tech Stack

- HTML
- CSS
- JavaScript

##  Purpose

Built to help freelancers and businesses quickly generate invoices.
